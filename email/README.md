# Default HTML Email Templates for PortSIP PBX

This repository contains the default HTML templates for PortSIP PBX email notifications.

To use these templates, copy and paste them into PortSIP PBX by following the guide below:  
**[Default HTML Templates](https://support.portsip.com/portsip-communications-solution/portsip-pbx-administration-guide/31-configuring-email-notifications/customizing-email-templates#default-html-templates)**

## Note

These templates are intended for PortSIP PBX v22.5.2 and earlier versions only.
